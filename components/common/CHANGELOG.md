# 로고 변경 이력

## v2.0.0 - 2025.11.04

### 🎨 디자인 완전 변경

**기존 (v1.0.0)**:

- 발바닥 모티브
- 중앙 패드 + 4개 발가락
- 심플한 디자인

**새로운 (v2.0.0)**:

- 카메라 + 강아지/고양이 모티브
- 사진 앱 컨셉 강조
- 더 친근하고 직관적인 디자인

### 주요 변경사항

1. **메인 모티브 변경**
   - 발바닥 → 카메라 모양
   - 사진 기록 앱임을 직접적으로 표현

2. **캐릭터 추가**
   - 강아지 실루엣 (왼쪽)
   - 고양이 실루엣 (오른쪽)
   - 중앙 작은 하트

3. **구조 개선**
   - 카메라 본체 (둥근 모서리)
   - 상단 돌출부 (뷰파인더)
   - 플래시 디테일
   - 흰색 렌즈 원

4. **그라디언트 제거**
   - 단색 버전으로 통일
   - 깔끔하고 모던한 느낌
   - 다양한 배경에 잘 어울림

### 파일 변경

- ✏️ `Logo.tsx` - 완전히 새로운 디자인
- ✏️ `AppIcon.tsx` - 카메라 버전으로 업데이트
- ✏️ `README.md` - 문서 업데이트
- 📝 `CHANGELOG.md` - 변경 이력 추가

### 마이그레이션 가이드

기존 코드를 변경할 필요 없습니다. API는 동일합니다.

```tsx
// 기존 코드 그대로 사용 가능
<Logo size={80} />
<Logo size={80} color="#FF9E80" />
<LogoWithText size={120} />
<AppIcon size={120} />
```
