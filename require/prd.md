## 1. 제품 개요

### 1.1 제품명

**멍냥일기 (Meong-Nyang Diary)**

### 1.2 제품 비전

반려동물과 함께하는 소중한 매일을 자동으로 정리하고, 온 가족이 함께 반려동물의 성장을 기록하며 추억을 공유하는 반려동물 전용 성장 기록 앱

### 1.3 목표 사용자

**Primary Target**: 반려동물을 키우는 2030세대 중 매일 사진을 찍지만 정리하지 못하고 갤러리에 방치하는 사람

**세부 페르소나**:

- 김멍멍 (28세, 여성, 회사원)
  - 2살 웰시코기 '보리' 양육 중
  - 하루에도 수십 장 사진 촬영
  - 갤러리에 3,000장 이상의 보리 사진 방치
  - 인스타에 반려견 계정 운영하지만 정작 갤러리 정리는 못함
  - 보리의 100일, 200일, 1살 등 기념일 사진을 찾기 어려움

- 박냥냥 (32세, 남성, 디자이너)
  - 1살 스코티시폴드 '쿠키' 양육 중
  - 외가, 친가 모두 쿠키를 손주처럼 아낌
  - 부모님께 사진 보내드리고 싶지만 일일이 카톡으로 보내기 번거로움
  - 쿠키 성장 영상을 만들어보고 싶지만 시간 없음

### 1.4 핵심 문제 정의

1. **정리의 어려움**: 매일 찍는 반려동물 사진이 갤러리에 방치되어 나중에 찾기 어려움
2. **기념일 관리 부재**: 입양일 기준 100일, 1살 등 중요한 날을 놓침
3. **공유의 번거로움**: 가족들에게 일일이 사진 공유하기 귀찮음
4. **추억 소실 위험**: 핸드폰 고장이나 분실 시 모든 사진 날아갈 수 있음
5. **성장 과정 시각화 어려움**: 우리 아이가 얼마나 자랐는지 한눈에 보기 힘듦

### 1.5 핵심 가치 제안

- **자동 정리**: 입양일 기준 날짜별 자동 사진 정리
- **가족 공유**: 온 가족이 함께 반려동물 성장 기록 공유
- **성장 시각화**: 자동 성장 영상 및 타임랩스 제작
- **안전한 보관**: 클라우드 백업으로 추억 보호
- **덕질 문화**: 우리 집 '최애'에 대한 덕질 플랫폼

## 2. 제품 목표

### 2.1 비즈니스 목표

- **3개월 내**: MAU 3,000명 달성
- **6개월 내**: 프리미엄 전환율 4% 달성
- **12개월 내**: 월 구독 수익 $3,000 달성

### 2.2 사용자 목표

- 반려동물 사진 찾는 시간 10분 → 10초로 단축
- 가족 공유로 반려동물에 대한 가족 참여도 2배 증가
- 월 1회 이상 성장 영상 시청으로 추억 회상

### 2.3 성공 지표 (KPI)

- **참여도**: 주간 활성 사용자(WAU) / 월간 활성 사용자(MAU) 비율 > 55%
- **리텐션**: D7 리텐션 > 35%, D30 리텐션 > 20%
- **사진 업로드**: 사용자당 주평균 사진 업로드 > 10장
- **가족 참여**: 가족 초대율 > 40%

## 3. 경쟁 서비스 분석

### 3.1 쑥쑥찰칵 (벤치마크 대상)

**강점**:

- 생후 날짜별 자동 정리
- 가족 공유 및 댓글 기능
- 80% 높은 전환율

**약점**:

- 육아 전용이라 반려동물에는 부적합
- 아이 중심 UI/UX

**우리의 차별점**: 반려동물 특화 (품종, 예방접종, 건강 기록 등)

### 3.2 일반 사진 앱 (구글 포토, 네이버 클라우드)

**강점**:

- 무제한 저장 공간
- AI 자동 분류

**약점**:

- 반려동물 특화 기능 없음
- 가족 공유 어려움
- 입양일 기준 정리 불가

**우리의 차별점**: 입양일 기준 타임라인, 가족 커뮤니티

### 3.3 반려동물 앱 (핏펫, 멍멍이 일기장)

**강점**:

- 건강 관리 기능
- 산책 기록

**약점**:

- 사진 정리에 최적화 안 됨
- 가족 공유 기능 약함

**우리의 차별점**: 사진 중심 기록 + 가족 덕질 문화

## 4. 기능 명세

### 4.1 MVP 핵심 기능 (Phase 1 - 8주)

#### 4.1.1 반려동물 프로필 등록 [구현 예정]

**기능 설명**:
첫 실행 시 반려동물 기본 정보 등록

**입력 항목**:

- 이름 (필수)
- 프로필 사진 (선택)
- 종류 (강아지/고양이 선택 필수)
- 품종 (선택, 드롭다운)
- 입양일 (필수) ← 가장 중요!
- 생년월일 (선택)
- 성별 (선택: 남/여/중성화남/중성화여)
- 몸무게 (선택)

**추가 기능**:

- 다반려가구를 위한 반려동물 여러 마리 등록 가능
- 프로필 전환 (사진 업로드 시 어떤 반려동물인지 선택)

**UX 요구사항**:

- 온보딩 과정에서 자연스럽게 입력 유도
- "나중에 입력하기" 옵션 제공하지 않음 (입양일은 필수)

**현재 상태**: 더미 데이터로 UI만 구현됨. API 연동 및 실제 등록 로직 필요.

**기능 설명**:
갤러리에서 사진 업로드 시 입양일 기준으로 자동 분류

**정리 방식**:

- **캘린더 뷰** [✅ 구현 완료]: 날짜별로 사진 표시
  - 7x5 그리드 캘린더
  - 원형 썸네일 + 날짜 오버레이
  - 월 네비게이션
  - 미래 날짜 비활성화
  - 파일: `app/(tabs)/index.tsx`

- **폴더 뷰** [✅ 구현 완료]:
  - D+100, D+200, D+300 기념일 폴더
  - 1개월, 2개월, 3개월... 월별 폴더
  - 1살, 2살 연도별 폴더
  - 2열 그리드
  - 파일: `app/(tabs)/calendar.tsx`

- **타임라인 뷰** [구현 예정]: 최신 순으로 스크롤

**자동 수집 정보**:

- 사진 촬영 날짜/시간 [구현 예정]

- 입양일 기준 D-day 자동 계산 [✅ UI 구현 완료, 더미 데이터]

**UX 요구사항**:

- 업로드 진행 상황 표시 [구현 예정]

**현재 구현**:

#### 4.1.3 AI 얼굴 인식 및 자동 분류 [구현 예정]

**기능 설명**:
갤러리에서 우리 반려동물 사진 자동 감지

**작동 방식**:

1. 프로필 사진으로 반려동물 얼굴 학습
2. 갤러리 스캔하여 해당 반려동물 사진 자동 추천
3. "이 사진들을 멍냥일기에 추가할까요?" 알림
4. 선택 후 일괄 업로드

**기술 구현**:

- 모바일 기기 내 On-device AI (프라이버시 보호)
- 또는 서버 기반 얼굴 인식 API 활용

**UX 요구사항**:

- 초기 학습 시 프로필 사진 외 2-3장 추가 학습 유도
- 오인식 방지를 위한 수동 확인 단계 포함

#### 4.1.4 가족 초대 및 공유 [✅ UI 구현 완료, API 연동 필요]

**기능 설명**:
온 가족이 함께 반려동물 사진을 보고 소통

**초대 방식**:

- 카카오톡, 문자 링크 공유 [구현 예정]
- 초대 코드 입력 [구현 예정]
- QR 코드 스캔 [구현 예정]

**권한 설정**:

- **관리자** (반려동물 주인): 모든 권한
- **가족**: 사진 업로드, 댓글, 좋아요 가능
- **읽기 전용**: 사진 보기만 가능

**가족 그룹 설정**:

- "우리 가족" (친가 + 외가 함께)
- "엄마 식구들"
- "아빠 식구들"
- 그룹별 댓글 비공개 설정 가능

**UX 요구사항**:

- 초대받은 사람은 앱 설치 후 바로 접근 가능
- 가족이 사진 올리면 푸시 알림

**현재 구현**:
- 설정 화면 내 "가족 관리" 섹션 추가
- "가족 구성원 보기" (4명 표시)
- "가족 초대하기" 버튼
- 더미 데이터로 UI 표시
- 파일: `app/(tabs)/settings.tsx`

**변경 사항**: 별도 "가족" 탭에서 설정 내 섹션으로 이동

#### 4.1.5 댓글 및 좋아요 [구현 예정]

**기능 설명**:
사진마다 가족들이 댓글과 좋아요 남기기

**기능 구성**:

- 사진별 댓글 (무제한)
- 좋아요 (중복 불가)
- 댓글 알림
- 이모지 반응 (❤️😊😂🐶🐱)

**댓글 비공개 기능**:

- "엄마 식구들만 보기"
- "아빠 식구들만 보기"
- 관리자는 모든 댓글 볼 수 있음

**UX 요구사항**:

- 댓글 입력 시 키보드 자동 올라옴
- 댓글 수정/삭제 가능
- 욕설 필터링

#### 4.1.6 성장 영상 자동 생성 [구현 예정]

**기능 설명**:
월별로 업로드한 사진들을 자동 편집하여 영상 제작

**생성 주기**:

- 매월 1일 자동 생성 (지난 달 사진 기반)
- 수동 생성 가능 (특정 기간 선택)

**영상 구성**:

- 업로드한 사진 시간 순 배열
- 사진당 1-2초 노출
- 배경음악 자동 삽입 (선택 가능)
- 오프닝: "보리의 2024년 12월"
- 엔딩: "총 150장의 추억"

**기술 구현**:

- 서버에서 영상 렌더링
- 또는 클라이언트에서 FFmpeg 활용
